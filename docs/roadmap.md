# Roadmap

只有代码、异常处理和验证完成后，任务才标记为已完成。

## 已完成：v1 MVP

- [x] React、Vite 和 Tailwind CSS 基础页面
- [x] 输入区、卡片列表、空状态、加载状态和错误状态
- [x] DeepSeek 术语提取和卡片生成服务端接口
- [x] 模型 JSON 输出与字段校验
- [x] localStorage 卡片读取、新增、去重、更新和删除
- [x] “看定义猜术语”复习模式
- [x] 答题统计、权重更新和按权重抽题
- [x] Vitest、React Testing Library 和 DeepSeek 错误映射测试
- [x] Vite 生产构建配置

## 部署验收

- [ ] 将仓库连接到 Vercel
- [ ] 配置 `DEEPSEEK_API_KEY`
- [ ] 在生产环境完成一次真实端到端请求
- [ ] 将线上地址和截图加入 README

## v1.1：使用体验

- [ ] 导入 `.txt` 和 `.md` 文件
- [ ] 从 URL 抓取并清理正文
- [ ] 支持卡片搜索、筛选和批量删除
- [ ] 导出和导入 JSON，提供本地备份
- [ ] 支持 Anki 可导入格式

## v2：能力扩展

- [ ] 英文技术写作辅助
- [ ] Prompt 版本选择与效果对比
- [ ] 记录失败样本，建立小型评估集
- [ ] 更完整的间隔重复算法和到期复习队列
- [ ] 在确认真实需求后评估账号和云同步

## 暂不考虑

- 日常场景词汇
- 社区、排行榜和社交功能
- 多用户数据库
- 多模型切换
- 无限制长文档处理
