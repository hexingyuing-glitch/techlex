## 预期问题与处理方式

### 问题 1：本地成功，生产环境 API 调用失败

- **原因**：CORS、环境变量、供应商浏览器调用限制或 Vercel 配置不同。
- **处理**：在部署预览环境验证真实请求，根据供应商规则决定前端直连还是服务端代理。

### 问题 2：README 与实际行为不一致

- **原因**：文档在开发前先写，命令、变量名或功能状态可能变化。
- **处理**：从空目录按 README 重走一次安装步骤，只保留已验证的命令、截图和在线地址。

### 问题 3：只验证理想路径

- **原因**：演示时容易忽略失败状态。
- **处理**：建立发布清单，至少覆盖输入、API、解析、存储和复习五类失败路径。
